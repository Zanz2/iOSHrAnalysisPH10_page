Frequently Asked Questions

[Split GPX] How can I get my route onto my GPS watch/navigator/device?

Open the files you exported using the "Files" app on iOS, then export them using the share button into the app you are using to connect to your GPS device (with garmin devices that is "Garmin connect".

[Split GPX] How can I split a route into asymetric chunks in Split GPX(different lengths per segment)?

To create asymmetric segments (different lengths), first split your route into larger segments. Then re-import specific segments and split them again. For example, if main_route.gpx is initially split into main_route_S1.gpx, main_route_S2.gpx, etc., you can further divide main_route_S2.gpx into main_route_S2_S1.gpx and main_route_S2_S2.gpx. Simply use these smaller segments in place of the original larger segment.

[Split GPX] Id like to stop seeing my location in Split GPX, how do I turn it off?

To disable/change location services for Split GPX: Go to Settings - Privacy and Security - Location Services. Scroll to find "Split GPX", click on it. Select "Never" to completely disable location access. You can also choose other options like "Ask Next Time Or When I Share" or "While Using the App" if you want limited location access instead of completely disabling it.

[Split GPX] Im dealing with an extremly large file (100+ Megabytes), Split GPX seems to be frozen / saving the segments takes a long time.

Using the traditional Map view speeds things up a bit, however when working with 1000+ segments saving them might take along time, please have patience. The app might seem frozen when loading the map or saving the segments but is actually just processing your request.
